# MongoDB Write Flow Visualizer

This project visualizes the write flow in MongoDB, allowing users to understand how data is written into the database.

## Getting Started

### Prerequisites
- MongoDB 
- Node.js (for running the project)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/vinodkrishnan23/mongodb-write-flow-visualizer.git
   ```

2. Navigate to the project folder:
   ```bash
   cd mongodb-write-flow-visualizer
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```
   
4. Start the application:
   ```bash
   npm start
   ```

### Usage
Follow the instructions on the web interface to visualize the write flow of MongoDB.
